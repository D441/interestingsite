1. Добавление классов +
+	1. Добавить дивам классы bad, good, simple. 
+	2. Добавить дивам цвет с помощью классов: bg-danger. bg-success, bg-warning 
+	3. Добавить класс для финиша - finish, для костей - cube, для денег - money
2. Найти элементы в js +
+	1. Найти дивы с помощью команды document.querySelectorAll()
+	2. Сохранить дивы в переменные be, ge, se;
+	3. Найти финиш, кости, деньги и сохранить в переменные.
3. Действия +
+	1. Если нажимаем на кость, то в alert выходит рандомное число.
+	   По нажатию сгенерировать рандомное число.
+	2. Нажатие на be, то -100 очков. Появление фишки с помощью backgroundImage
+	3. Нажатие на ge, то +100 очков. Появление фишки с помощью backgroundImage
+	4. Нажатие на se, то +10 очков. Появление фишки с помощью backgroundImage
+	5. Нажатие на финишю Если меньше 50 очков, то проиграл, иначе выйграл.
